MEU PORTFÓLIO
Sobre Mim Projetos

Este portfólio foi desenvolvido como parte das atividades da disciplina de Desenvolvimento Web, do curso Técnico em Desenvolvimento de Sistemas – 2º ano. O objetivo deste trabalho é apresentar os conhecimentos adquiridos durante as aulas, demonstrando a criação de páginas web utilizando HTML e CSS. Ao longo do ano, novos projetos desenvolvidos em aula serão adicionados neste portfólio, permitindo acompanhar minha evolução na área de desenvolvimento web.
